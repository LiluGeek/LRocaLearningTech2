This presentation discusses the differences and similarities between assessment in online instruction versus assessment in face-to-face (F2F) instruction.

This instructional video is part of a course I designed on the fundamentals of web design. In this module the learners code an index.html page.

 

 

HOW TO HARVEST SEEDS FROM A TOMATO
